# JiE-sport
该网站旨在帮助校内师生更好的了解校内多彩的体育生活，使时效性较强的体育活动的宣传更及时有效，并将报名一体化、信息化、自动化，补上了以前人力一一逐个统计报名信息并容易遗漏的缺点，极大的解放了生产力，将部门同学从重复性劳动中解放出来，  像类似运动会此种大型项目，报名人数多，项目种类繁杂，如何确保更多的同学及时了解活动并在截止时间前参与进来，  网站设计现代化，更符合现代审美，采用element-plus组件进行设计开发，设计多借鉴当前主流大网站，如vue文档、element-plus文档，  网站结构层次清晰，采用导航栏区分不同类别的用户信息与交互，页面层级之间交互良好
